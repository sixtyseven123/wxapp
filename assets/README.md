# 资源文件说明

此文件夹用于存放静态资源文件。

## 文件夹结构

```
assets/
├── icons/           # tabBar图标（需要手动添加）
│   ├── home.png             # 首页图标
│   ├── home-active.png      # 首页选中图标
│   ├── history.png          # 历史记录图标
│   ├── history-active.png   # 历史记录选中图标
│   └── about.png            # 关于图标
│   └── about-active.png     # 关于选中图标
│
└── images/          # 其他图片资源
    ├── share.png            # 分享图片
    ├── empty-history.png    # 空状态图片
    └── logo.png            # logo图片
```

## 图标要求

### tabBar图标
- 尺寸：81×81 像素
- 格式：PNG
- 建议使用透明背景

### 分享图片
- 尺寸：500×400 像素
- 格式：PNG/JPG

## 快速生成图标

可以使用在线工具生成图标：
1. https://www.iconfont.cn/ - 阿里图标库
2. https://www.flaticon.com/ - 免费图标资源

或者使用微信开发者工具自带的设计工具。
